# 访问修饰符

* 可以修饰属性和方法的访问范围。

  | **访问修饰符** | **本类** | **同包** | **子类** | **其他** |
  | -------------- | -------- | -------- | -------- | -------- |
  | **private**    | y        |          |          |          |
  | **默认**       | y        | y        |          |          |
  | **protected**  | y        | y        | y        |          |
  | **public**     | y        | y        | y        | y        |
